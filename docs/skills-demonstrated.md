# Skills Demonstrated

## Remote Access
- Logging into remote systems using SSH  
- Understanding secure authentication workflows  

## Filesystem Enumeration
- Listing files (`ls`)
- Revealing hidden files (`ls -a`)
- Identifying file types (`file`)

## File & Directory Management
- Creating files (`touch`)
- Creating directories (`mkdir`)
- Copying (`cp`)
- Moving or renaming (`mv`)
- Removing files and directories (`rm`, `rm -r`)

## Permissions & Ownership
- Reading symbolic permissions (`rwxr-xr--`)
- Converting symbolic to numeric (e.g., `754`)
- Understanding owner, group, and other permissions

## User Management
- Switching users (`su`)
- Using login shells (`su -l`)

## System Awareness
- Understanding `/etc`, `/var`, `/root`, `/tmp`
- Recognising where attackers commonly hide files
